# Musings — Agent Guidance

Always commit and push musings file changes. Musings are lightweight, stream-of-consciousness notes that capture raw ideas before they become formal artifacts.

When creating or editing a musing:
1. Write freely — no fixed format required
2. Commit immediately after writing
3. Push to the remote
4. If a musing solidifies into a plan, ADR, or spike, move the content out and leave a pointer

See `~/.agents/agents-md-detail/locations/musings.md` for full conventions.
